Request¶

class pyvdp.request.VisaRequest(resource, api, method, http_verb, query_string='', data='', auth_method='ssl', headers=None, config=None)[source]¶

Constructs a request to Visa API to provided resource, method and http verb.

After request is constructed, it shall be submitted with send() method, which returns a requests.Response object.

This class is not assumed to be instantiated on its own (consider it abstract). Instead, every VISA API implementation must provide its own request class, inheriting from VisaRequest and providing a corresponding VISA API resource name as a resource argument value (see visa.visadirect.request as an example)

Parameters:
  • resource (str) – Required. VISA API resource name
  • api (str) – Required. VISA API api name
  • method (str) – Required. VISA API endpoint method
  • http_verb (str) – Required. VISA API HTTP verb (GET, POST, PUT)
  • query_string (str) – Conditional. Query string to append to API endpoint
  • data (object) – Conditional. Data payload for POST requests, VisaTransacton object
  • auth_method (str) – Conditional. Authentication method. Possible values are: ssl - for certificate-based authentication (default) or token - for x-pay-token authentication
  • headers (dict) – Optional. Additional headers.
send()[source]¶

Submits a data object or query string id to VISA using self.api_endpoint field and corresponding http verb.

Returns:result: Resulting dictionary.
