Usage¶

A collection of functional calls for VisaDirect mVISA APIs, including:

  • CashInPushPayments (GET, POST)
  • CashOutPushPayments (GET, POST)
  • MerchantPushPayments (GET, POST)

Data model for mVISA CashinPushPayments API.

  • stan (int) – Required. Systems trace audit number. 6 digits positive integer.
  • recipient_pan (str) – Required. Recipient primary account number (PAN). 13-19 characters string.
  • amount (float) – Required. Transaction amount. 12 digits, 3 fractions.
  • card_acceptor (CardAcceptor) – Required. Instance of CardAcceptor() data object
  • transaction_currency_code (str) – Required. Transaction ISO currency code. 3 characters string.

PurchaseIdentifier¶

class pyvdp.visadirect.mvisa.data.PurchaseIdentifier(**kwargs)[source]¶

Data model for VisaDirect mVISA PurchaseIdentifier object.

Used as a part of MerchantPushPaymentTransaction().

Parameters:
  • type (str) – Required. Primary ID as defined by mVISA. 1 character string.
  • reference_number (str) – Required. Key data element for matching a message to others within a given transaction set. Value will be the same as what has been provided in the request. Max 13 characters string.
