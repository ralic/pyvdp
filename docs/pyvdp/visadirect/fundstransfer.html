VisaDirect FundsTransfer¶

See also

https://developer.visa.com/products/visa_direct/reference#visa_direct__funds_transfer

Usage¶

A collection of functional call for VisaDirect FundsTransfer APIs, including:

  • PullFundsTransactions (GET, POST)
  • MultiPullFundsTransactions (GET, POST)
  • PushFundsTransactions (GET, POST)
  • MultiPushFundsTransactions (GET, POST)
  • ReverseFundsTransactions (GET, POST)
  • MultiReverseFundsTransactions (GET, POST)

PullFunds¶

pyvdp.visadirect.fundstransfer.pull_funds.get(status_id, multi=False)[source]¶

Fetches a status of previously submitted PullFundsTransaction() by transaction identifier.

Parameters:
  • status_id (str) – Required. Transaction status identifier.
  • multi (bool) – Conditional. Indicates that transaction is a batch (MultiPullFundsTransaction())
Returns:

Dictionary with VDP API response
Example:
from visa.visadirect.data.transaction import PullTransaction, MultiPullTransaction
from visa.visadirect.fundstransfer import pull_funds
from visa.exceptions import VisaTimeoutError

ca = CardAcceptor(name='Acceptor 1',
                  country='RU',
                  terminal_id='TID-9999',
                  id_code='CA-IDCode-77765')

atom = PullFundsTransaction(stan=123456,
                            amount=123.45,
                            sender_pan='12345678123456678',
                            sender_card_expiry_date='12-2020',
                            sender_currency_code='USD',
                            card_acceptor=ca)

# Let's pretend its a batch
transactions = [atom]
mpt = MultiPullFundsTransaction(transactions=transactions)

try:
    # This will return HTTP 202 (multi always does)
    pull_funds.send(transaction=mpt, multi=true)
except VisaTimeoutError as e:
    # Grab an exception message and use it as an argument
    result = pull_funds.get(status_id=e.message)

                           approval_code='20304B',

                          ode=ode,
                          sender_pan='1234567812345678',

                          transaction_identifier=12345,
