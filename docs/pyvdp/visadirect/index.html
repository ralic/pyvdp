class pyvdp.visadirect.data.VisaDirectTransactionBatch(transactions)[source]¶

Abstract TransactionBatch object model.

Constructs and populates a ‘transaction of transactions’, normally submitted with multi requests. This class is not supposed to be instantiated on its own, it is just providing a shell for batch requests

Parameters:transactions (list) – Required. List of VisaDirectTransaction() objects.
