exception pyvdp.exceptions.VisaTimeoutError(result)[source]¶

Raised when POST request is timed out.

This exception is somewhat special, since while being a timeout error, it is in fact a standard response in case multiple transactions were submitted, e.g. through pullfunds.send(multi=True) call.

It returns 202 HTTP code which indicates, that request is received and being processed under the hood, the content is a transaction ID which can be used in get requests to get a status on this specific submission.

Note

  • VISA API HTTP Code: 202
  • VISA Error Code: None
