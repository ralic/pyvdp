Funds Transfer Inquiry¶

See also

https://developer.visa.com/products/paai/reference#paai__paai

The Funds Transfer Attributes Inquiry API is often used with a funds transfer to/from a Visa payment account to determine key characteristics of a recipient card before initiating the transfer like country, card-type, block status, etc. The API has been enhanced to query PullFundsTransaction and PushFundsTransaction eligibility (both domestic and cross-border) for multiple U.S. debit networks.

Funds Transfer Inquiry API retrieves the attributes, which determine the key characteristics of a recipient payment account before initiating a funds transfer, by providing the Primary Account Number (PAN)

Usage¶

pyvdp.paai.fundstransferattinq.cardattributes.fundstransferinquiry.send(data)[source]¶

Submits Funds Transfer Inquiry request.

Parameters:data – Required. Instance of FundsTransferInquiryData()
Returns:Dictionary with VDP response
Example:
from visa.paai.fundstransferattinq.cardattributes.data import FundsTransferInquiryData
from visa.paai.fundstransferattinq.cardattribute import fundstransferinquiry

data = FundsTransferInquiryData(stan=123456,
                                rrn="330000123456",
                                pan="4957030420210512")

result = fundstransferinquiry.send(data=data)

Data Objects¶

class pyvdp.paai.fundstransferattinq.cardattributes.data.FundsTransferInquiryData(stan, **kwargs)[source]¶

Funds Transfer Inquiry data object model.

Parameters:
  • pan (str) – Required. Primary account number (PAN). 13-19 characters string.
  • stan (int) – Required. Systems Trace Audit Number. 6 digits integer. Generated by payment application, unique within the scope of application until expired (usually within 24 hours). Can be reused if more than 999 999 transactions were performed during a day.
  • rrn (str) – Optional. Retrieval Reference Number. 12 digits string. If not provided, will be generated as a function from STAN.
  • acquiring_bin (int) – Optional. BIN under which FundsTransfer app is registered. 6-11 digits integer.
  • acquirer_country_code (int) – Optional. 3 digits acquirer country code.
Example:
{
    "primaryAccountNumber": "4957030420210512",
    "retrievalReferenceNumber": "330000550000",
    "systemsTraceAuditNumber": "451006"
}
